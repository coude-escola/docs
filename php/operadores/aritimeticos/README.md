### -> Operadores Aritiméticos
- Adição
  ```php
    echo 8 + 8;
    # 16
  ```
- Subtração
  ```php
    echo 7 - 5;
    # 2
  ```
- Multiplicação
  ```php
    echo 3 * 25;
    # 75
  ```
- Divisão
  ```php
    echo 144 / 12;
    # 12
  ```
- Resto da Divisão
  ```php
    echo 345678 % 2;
    # 0

    # NOTA -> todo número, onde o resto da divisão por 2 for igual a zero, é par.
  ```
- Potências
  ```php
    # 3^2

    echo 3 ** 2;
    # 9
  ```
- Raiz
  ```php
    # 2√16

    echo 16 ** (1/2);
    # 4
  ```
